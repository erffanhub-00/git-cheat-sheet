## ✨ ویژگی‌ها

- 📚 بیش از ۴۰ دستور گیت با توضیح و مثال
- 🔍 جستجوی سریع
- 🏷️ فیلتر بر اساس سطح (مقدماتی/پیشرفته/شاخه/ریموت)
- 📋 کپی کردن دستور با یک کلیک
- 🌙 تم تاریک/روشن
- 📱 کاملاً واکنش‌گرا
